# Google Sheets Formulas

## Schedule Reports

```=SCHEDULEREPORTS(frequency);```

**Frequencies**

* Daily
* Weekly
* Monthly
* Quarterly
* Yearly

## Get URL Information

```=GETPREFERREDURL(rootDomain, format);```

**Formats**

* Spliced URL (Separated into Columns by Components)
* Combined URL (All Components Combined into Single Cell)

**Components**

* Connection
* Protocol
* Subdomain
* Root Domain
